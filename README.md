# E-Commerce Sales Analysis using SQL & Excel

##  Project Overview
This project focuses on analyzing e-commerce sales data to gain meaningful business insights.
SQL was used to query and analyze the data, and Excel was used to create pivot tables and an interactive dashboard.

##  Tools & Technologies
- MySQL
- Microsoft Excel
- GitHub

## Key Analysis Performed
- Total Gross Sales and Net Sales
- Category-wise Order Analysis
- Customer Age Group Analysis
- Location-wise Order Distribution
- Discount Analysis

## Dashboard
An Excel dashboard was created to visually represent:
- Sales performance
- Category distribution
- Customer demographics

##  Project Files
- `ecommerce_analysis.sql` – SQL queries used for analysis  
- `Ecommerce_SQL_Analysis.xlsx` – Excel file with pivots and dashboard  
- `dashboard.png` – Dashboard screenshot  
- `category_pivot.png` – Category analysis pivot screenshot  

##  Conclusion
This project demonstrates how SQL and Excel can be effectively used together
to analyze large datasets and support data-driven business decisions.
